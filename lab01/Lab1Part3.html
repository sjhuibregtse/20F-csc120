Part 3: Declare, Instantiate, and Use objects from existing Java classes.

  1. Watch this video, which explains how to declare, instantiate, and use objects from an existing class.

  2. Create your Lab1 project:

    1. Create a new NetBeans project called Lab1.

    2. Visit the CSC 120 Resources web page,
          Right-click on the link to MUPanel.java in the Starter Templates section of the page,
          Choose "Save Link As..." or "Save File As...",
          and save the file in your CSC120\Java\Lab1\src folder:

    3. Again on the CSC 120 Resources web page,
          Left-click on the link to MU Java Classes in the JavaDocs and Java Classes section at the bottom of the page,
          Right-click on the link to Flag.java on the MU Classes page,
          Choose "Save Link As..." or "Save File As...",
          and save the file in your CSC120\Java\Lab1\src folder

    4. Before moving to the next step in this lab, be sure that your Lab1 project contains two files in the src folder that are named Flag.java and MUPanel.java.

  3. Declare a Flag object in the appropriate place in the MUPanel.java file, like this:
    private Flag flag1;
  4. In the appropriate place in the MUPanel.java file instantiate (construct) your Flag object. To do this you'll need to know what parameters to pass to the constructor method of the Flag class. To find that out you can click on the MU Java Docs page for Balloon, Flag, etc. from the CSC 120 Resources page, and from there click on the docs for the Flag class. Now scroll down and find the description of the constructor method to see what parameters it expects. You'll need to add a line of code in the MUPanel.java file in the space where you instantiate objects. It should look something like this: 
    flag1 = new Flag( ... parameters go here ... );
    There are 4 values that you must specify to instantiate a new Flag object: you must first specify 2 Integer values that represent the screen position of the upper-left corner of the Flag, and then you specify two different Color values; one Color for the Flag's background, and a second Color for the horizontal stripe across the middle of the Flag.  The parameters in parentheses are always separated by commas.

  5. Finally, in the paintComponent method of MUPanel, add a line of code that calls the draw method of your Flag object. Don't forget to pass it the Graphics object it expects.

  6. Now declare 3 more Flag objects, instantiate them, and ask them to draw themselves. Make sure your 4 Flag objects each use different flag & stripe color combinations, are in different locations, and are visible (not outside of the MUPanel window).  Here is a list of color names you can use in Java.

  7. Change the String used in the "setName" method of the MUPanel constructor so that your name replaces Samantha's in the title of the output window when the program is run.  Next, edit the comment header at the top of the MUPanel class to show your name and today's date instead of Samantha Student and the words "Today's Date". Finally, add a brief description of what your program does.

  8. Run your program so that the output window appears on your screen.  Take a screen capture of just the output window that includes the title bar showing your name (in Windows, this can be done in one of two ways:

    1. Click on the output window so that it is selected, then use the [Alt]-[PrtSc] keystroke.  Next, open Windows Paint (type "Paint" in the Windows search area), paste in your screen capture, and save the image as a .png file

    2. Use the Windows Snip & Sketch tool (search for "Snip" in the Windows search area), then save your screen snip as a .png file

    Ask your lab instructor for help if needed with this process.  You will save your screen capture as a .png file to be submitted along with your NetBeans project.
