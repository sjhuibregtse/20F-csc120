Download and Install NetBeans 12 on Your Own Computer

These instructions will guide you through the steps needed to install OpenJDK 11 and Apache NetBeans 12.0 on your computer.

OpenJDK is the Java Development Kit, and is needed by NetBeans in order for it to function properly.  You must install OpenJDK first, then install NetBeans.

Choose the type of computer you have:

  • Instructions for installing on a PC running Windows.

  • Instructions for installing on a Apple Mac running OS/X.
