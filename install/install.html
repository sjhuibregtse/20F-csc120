Follow at least one set of the following instructions to prepare your workstation for this course:

  • Install Java JDK and NetBeans on your personal computer.


  • Use a web browser on your own computer or on a networked computer on campus to access the CS virtual desktop that has NetBeans already installed and that you will be able to use to complete your weekly coursework.

  • To access the CS Virtual Desktop from a location off-campus, request VPN access and use the CS Virtual Desktop


First-Year Students: Complete this request form for storage on the Mount Union Network

Mount Union students can have a personal storage space on the Mount Union network that can be used to store files, documents, programs, etc.  We encourage you to use this space as it is a convenient way to store your work for CSC 120 and other courses. Starting with the Fall 2020 semester, incoming students must apply for this storage space by filling out a form.  You should do this right away so that you can have this space for your work as soon as possible after the semesters starts.

If you are a returning student who entered Mount Union prior to Fall 2020, you do not have to apply for storage; this policy only applies to students who enter after the Spring 2020 semester.
