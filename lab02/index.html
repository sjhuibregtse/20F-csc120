CSC 120 Lab 2

In this lab we will:
  • Practice drawing rectangles, ovals, lines, and arcs using methods of the Graphics class.
  • Practice organizing objects in the viewer, specifying over & down coordinates, widths, heights, and arc-angles.
  1. Examine the Java Graphics Tutorial on the CSC 120 Resources Page.

    1. Java provides basic drawing capabilities via methods defined for the Graphics class.

    2. Visit the Java Graphics Tutorial page to see examples of several of the most-commonly used drawing methods.  We will use some of these methods in today's lab activity.

  2. Create a new NetBeans project called Lab2, using a special starter program that draws one rectangle.

    1. In NetBeans choose File | New Project | Java Application, etc.

    2. Make sure the Project Location is in your H:\CSC120\Java folder, and the Project Name is Lab2 (no spaces in Lab2).

    3. Right-click on the following link:  MUPanel.java, and then choose Save link as.... Save this file in your Lab2 src folder.

    4. Double-click on the MUPanel.java file to edit it. Look over the comment header at the top, and change the name and date to your name and today's date.

    5. Notice in the paintComponent method of the MUPanel class definition where we have used the g.setColor method to make the current drawing color be BLACK, and then used the g.drawRect method with parameters.

    6. Run the program to see what it does.

  3. Edit the MUPanel.java file to finish the House.

    1. We're going to use several more methods of the Graphics class to complete our house. We want the finished house to look approximately like this one.

    2. In the paintComponent method of the MUPanel class, we will now add code to draw the door and window for the house.  It is strongly recommended that you draw a picture on paper and label coordinates & widths & heights to figure out values for your over, down, width, and height for your new rectangles. Use Color.BLUE for the color of the door and Color.YELLOW for the color of the window (the light is on in the room).

      Consult the Java 8 Class Documentation to learn about the drawLine, drawRect, fillRect, and fillOval methods.

    3. Notice that the window has a black border around the outside of the window – to produce this effect in your program, you'll need to draw a BLACK rectangle after first filling a YELLOW rectangle for the window.

    4. Run the program, fix any errors that are noted, etc. until you get it to run properly, showing all three Rects in the output window.

    5. Use the fillOval method to paint an ORANGE Oval for the door knob, and use the drawLine method to draw horizontal and vertical lines to divide the window into panes, and to draw two diagonal lines for the roof.  Make all of the lines BLACK.

    6. Run the program, fix any errors that are noted, etc. until you get it to run properly, and your house looks approximately like this one.

  4. Make a tree for the yard, and a happy-face sun for the sky.

    1. We want the finished picture to look approximately like this one.

    2. Use methods from the Graphics class to fill a brown rectangle and a GREEN oval to make the tree.  For brown instead of using Color.WHATEVER use new Color(102, 51,0), which will create a new color. So, this line of code looks something like this:
      g.setColor( new Color(102, 51, 0) );

      Aside: If you want to create other colors that are not "built-in" to Java, use this link to the Decimal RGB Color Codes web page, which shows a color palette with numbers for the RGB (red-green-blue) values to pass to the Color constructor.

    3. Use a YELLOW Oval, two ORANGE Ovals, and one ORANGE Arc to make a happy-face sun.  Consult the Java 8 Class Documentation to learn about the drawArc method.

    4. Run the program, fix any errors that are noted, etc. until you get it to run properly, and your picture looks approximately like this one.

  5. Hand in your Lab2 Project.

    1. Ensure that the comment header at the top of the MUPanel.java file contains your name and today's date.  Also your name should appear in the title of the frame window, in place of Samantha Student.
    2. Run your completed program and save a screen capture as a .png file using the Windows Snip & Sketch tool or the [Alt]-[PrtSc] keystroke as you did for Lab # 1.
    3. Use File Explorer to convert the H:\CSC120\Java\Lab2 folder into a .zip file by right-clicking on the Lab2 folder and choosing "Send To > Compressed (zipped) folder".
    4. Then log into the D2L homepage, go to CSC 120 in your list of courses, choose "Assessments | Dropbox", and upload/submit the Lab2.zip and the screen capture .png files into the Lab2 dropbox using the techniques from Lab1.  Ask your instructor for help if you need it.

  6. If time remains in the class period:

    1. Experiment with other colors and drawing methods to add some extra features to the scene you have drawn for your Lab2 project.  Use your creativity — can you add grass?  a swing hanging from the tree?  stick figure people or dogs in the yard?  Godzilla's foot about to stomp on the house?  Etc., etc.
