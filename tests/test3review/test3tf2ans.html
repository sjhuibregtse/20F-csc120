True/False Question # 2 Answer:

Consider this Java program code: 
    String[] values;
    values = new String[420];
True or False: The smallest legal subscript value for the array named values would be 0.
true
false
Neither of these
The first element of every array always has subscript 0, regardless of the number of elements contained by the array.

By the way, for a true/false question, the answer can NEVER be "Neither of these", so that was a trick.
