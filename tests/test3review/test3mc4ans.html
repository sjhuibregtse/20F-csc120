Short Answer Question # 4:

a)   What sequence of values for the variable num is generated by the following loop heading? 
        for ( Integer num = 375;  num >= 200;  num = num - 25 )
For the part a) problem, num starts at 375 and goes down by 25, as long as its value is >= 200.  Therefore, the sequence of values generated by this loop would be:
375     350     325     300     275     250     225     200


b)   What sequence of values for the variable num is generated by the following loop heading? 
        for ( Integer num = 375;  num > 200;  num = num - 25 )
For the part b) problem, the use of the Boolean expression num > 200 instead of num >= 200 means that the loop will be executed one fewer time than the part a) loop.  In this case, the sequence of values generated by the loop would be:
375     350     325     300     275     250     225
