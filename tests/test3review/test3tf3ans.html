True/False Question # 3 Answer:

Consider this Java program code: 
    String[] values;
    values = new String[420];
True or False: The largest legal subscript value for the array named values would be 420.
true
false
Neither of these
The last element of any array always has a subscript that is one less than the length of the array.  So in this case, the subscript of the last element in the array is 420 – 1 = 419.
