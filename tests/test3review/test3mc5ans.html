Short Answer Question # 5:

Write a for loop heading that makes the control variable of the loop take on the following sequence of values, in this order:
120     151     182     213     244     275
The loop control variable must start with the value 120, increase each time through the loop by 31, and not exceed 275.  So, a for loop that would have its variable take on that sequence of values would be: 
     for ( Integer x = 120;  x <= 275;  x = x + 31 )
IMPORTANT NOTE:  You cannot have this code shown in red text below in the loop heading for this problem: 
     for ( Integer x = 120;  x <= 275;  x + 31 )
If you do that, x will never be changed, and will always be 120 in an infinte loop.  You must change the value of x each time through the loop with a statement like 
     x = x + 31
or 
     x += 31
as the third expression in the for loop.
