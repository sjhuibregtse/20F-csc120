Multiple Choice Question # 1 Answer:

Which of these is a legal Java variable name?
3Sally
isDivisibleBy2
sally smith
sally'sC
All of these are legal variable names
Variable names in Java cannot begin with a digit, cannot contain spaces, and cannot contain apostrophes.  Legal variable names start with a letter, and may contain letters, digits and/or underscores (the _ character) after the first letter.
