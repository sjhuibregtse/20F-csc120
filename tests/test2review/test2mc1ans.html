Multiple Choice Question # 1 Answer:

Which of these is a correct Boolean expression in Java that will be true when an Integer variable called aNumber is in the range from 1 to 2000, inclusive?
aNumber > 1 && aNumber < 2000
aNumber >= 1 && aNumber <= 2000
aNumber > 1 || aNumber < 2000
aNumber >= 1 || aNumber <= 2000
aNumber < 1 && aNumber > 2000
aNumber <= 1 && aNumber >= 2000
aNumber < 1 || aNumber > 2000
aNumber <= 1 || aNumber >= 2000
None of these are correct
The first choice doesn't include 1 and 2000 (that's called the range from 1 to 2000, exclusive), the third and fourth choices are ALWAYS true, the fifth and sixth choices are ALWAYS false, and the seventh and eighth choices are true when aNumber is NOT between 1 and 2000.
