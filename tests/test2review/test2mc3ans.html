Multiple Choice Question # 3 Answer:

There are two kinds of methods we can write in Java class definitions:  (1) void methods, and (2) non-void methods (also known as type-returning methods).  An example of the heading of a void method would be: 
    public void setTaxFilingStatus( String statusString ) {
An example of the heading of a non-void method would be: 
    public Double calculateAverage( Double firstNum, Double secondNum, Double thirdNum ) {
Question:  What kind of method is required to have a return statement in the body of the method?
void methods only
non-void methods only
both void and non-void methods must have a return statement
neither kind of method must have a return statement
A return statement is the mechanism used in Java to send the answer computed by a non-void method back to the place where the method was invoked.  If a method returns a value, it must have a return statement.  A void method ends when the last statement inside the braces defining the method body is executed.
