Multiple Choice Question # 2 Answer:

Consider this Java program code: 
    Integer one, two;
    one = 2;
    two = 1;
	
    two = two + one;
    one = two + one;
	
    System.out.println("one = " + one + " and two = " + two);
What output is displayed in the Java Console when this code is executed?
one = 1 and two = 2
one = 2 and two = 1
one = 3 and two = 3
one = 3 and two = 4
one = 3 and two = 5
one = 4 and two = 3
one = 5 and two = 3
None of these
Whenever a variable appears on the right-hand side of an assignment statement, Java always uses the most recent value that has been stored in that variable.  So in the last assignment statement above, two has the new value 3, which is added to the value of one, which is 2.
