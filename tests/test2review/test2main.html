  1. Consider the following Java class: 
        public class Book {
	
        private String title;
        private Integer numPages;
		
        public Book( String t, Integer n ) {
            title = t;
            numPages = n;
        }
		
        public String toString() {
            return "This book's title is " + title + " and it contains " + numPages + " pages";
        }
		
        public String getTitle() {
            return title;
        }
		
        public Integer getNumPages() {
            return numPages;
        }
		
        public void setTitle( String t ) {
            title = t;
        }
		
        public void setNumPages( Integer n ) {
            numPages = n;
        }
		
    } // end of class Book
    1. Write one line of code that declares a Book object named novel.

    2. Write one line of code that instantiates novel.

    3. Write one line of code that uses the toString method to print novel's data to the Java Console.

    4. Write one line of code that displays novel's title in the Graphical Output Window at location (300, 50).

    5. Write one line of code that sets novel's number of pages to 234.

    6. Write one line of code that prints novel's number of pages to the Java Console.

    7. Write a method named showNum, which accepts one Integer parameter and returns nothing. The method should print "big" to the Java Console if the value of the parameter is greater than 100, and otherwise the method should print "little" to the Java Console.
